# Capstone Movies Recommendation System

## Installation

Install Capstone Project 

1. Clone the repo
   ```sh
   git clone https://github.com/febrian2108/Capstone-Project.git
   ```
2. Go to project
   ```sh
   cd Capstone-Project.git
   ```
3. Run Frontend 
   ```sh
   cd frontend
   npm install
   npm run dev

   ```
4. Run Backend
   ```sh
   cd backend
   npm install
   npm run dev
   ```
5. Run Machine Learning
   ```sh
   cd ml-model
   python train_model.py
   ```
6. Go to website
   ```sh
   http://localhost:5173
